# Glossary Rules

统一术语：

1. `Shared Engineering Brain`：系统最终形态
2. `AI Engineering Operating System`：产品级定位
3. `Memory`：错误、复盘、规则、自我进化
4. `Standards`：工程规范、TDD、DRY、文档规则
5. `Planning`：Harness Engineering、Spec Kit、task graph、execution plan
6. `Validation`：docs-check、contract-check、test-check、doctor、policy-check

命名要求：

1. 不要把整个系统重新简化回 `shared error memory`
2. 旧定位如仍需保留，只能作为历史阶段描述

# Deprecated Plans

本目录用于存放已明确废弃、且不再执行的 planning bundles。

这是一个 planning bundle 根目录说明文件。

移动规则：

- 目标已经被其他实现路径替代
- 需求已经失效，不再属于当前产品范围
- 即使代码已完成，也不应继续维护或推进

与 `archive/` 的区别：

- `archive/`：计划已完成，保留为历史交付记录
- `deprecated/`：计划已失效，保留为历史决策记录

当前状态：

- 截至 2026-04-08，审查 `docs/plans/active/` 后暂无明确废弃项。

# Planning Governance Gate

这是把 `plan-check` 引入 Agents-Memory 的 planning bundle。

目标是把 `plan-init` 生成出来的 planning 工件正式纳入校验门禁，让 Planning 层从“可生成”升级成“可治理”。

建议使用顺序：

1. 先写 `spec.md`
2. 再补 `plan.md`
3. 再确认 `task-graph.md`
4. 最后在 `validation.md` 里写最小验证路线

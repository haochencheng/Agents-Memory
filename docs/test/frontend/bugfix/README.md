# Frontend BugFix 记录

## 文件命名规范

```
{YYYYMMDD}-bug{NNN}-{简述}.md
```

示例：`20260408-bug001-overview-api-error.md`

## Bug 记录模板

```markdown
# Bug {NNN}: {简述}

- **发现时间**: 2026-04-08
- **页面**: /
- **严重级别**: high / medium / low
- **状态**: open / fixed

## 现象
截图 + 错误描述

## 根因
分析

## 修复方案
代码改动

## 验证
修复后截图
```
